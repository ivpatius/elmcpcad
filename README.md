# ElectroCAD AI

Интеллектуальная система для работы с электрическими схемами через ИИ (Ollama qwen3/qwen3-vl).

## Возможности

- 🔍 **Анализ схем**: Распознавание компонентов на фото/сканах через qwen3-vl:30b
- 📦 **Извлечение блоков**: Автоматическое выделение функциональных модулей
- 🤖 **Генерация схем**: Создание новых схем из текстовых требований
- 🔧 **Модификация**: Изменение существующих схем через ИИ
- 📚 **Библиотека**: Управление коллекцией электрических блоков
